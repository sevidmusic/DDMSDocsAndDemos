-- Define the App's dynamic output files in the App Package's DynamicOutput directory --

Note: The HtmlHead.php dynamic output file's content contains a <link> tag that references a css
      file, styles.css, this css file must be created in the App Package's css
      directory.

6.a Create css/styles.css:
    Run: vim "$HOME/SingleAppWebsite/DynamicOutput/HtmlHead.php"
    Add:
         body {  background: #000000; color: #ddf1ff; }
         .main-menu-container { list-style-type: none; }
         .main-menu-item { list-style-type: none; }
