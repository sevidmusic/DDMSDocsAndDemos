Note: Make sure to also create any css or js files referenced in any html defined
      in the dynamic output files. Css files should be placed in the App Package's
      css directory, js files should be placed in the App Package's js directory.
      This must be done so dsh knows to include them with the files copied from
      the App Package to the App whenever dsh --make-app is run to make an
      instance of the App.
